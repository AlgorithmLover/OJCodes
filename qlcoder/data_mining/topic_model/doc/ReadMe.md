#文本分类
##文本预处理
- 仅仅包含英文的文本处理

- 长度较短的文本
    - 仅仅包含链接的

- 噪音去除
    - 噪音词/可能无关词
    - badcase等提示词汇
    - 上一页，下一页，转义字符 `&nbsp;`，等html标记相关

- 无关信息去除
    - 图文来源
    - 编辑身份内容
    - 文章来源报社
    - 作者/记者/报社
    
- 不同类型的文本
    - NBA直播的内容
    - 小说的内容
    - 习题的内容
    - 简单摘要的内容
    - 统计报告
    - 与文章内容无关的人名
    - 快照
    - 信件类似
    - 访谈类型
    - 同一句话反复复制
